Cloud Database Settings

This system can sync to a Supabase (Postgres) cloud database. Paste your Project URL and anon key below.

For sign-in / sign-out, use the Account button.

Required database table 
Table name: documents
Columns:
- id (uuid, primary key)
- kind (text)              -- "forward" or "received"
- dts_no (text)
- from_office (text)
- details (text)
- received_by (text)
- to_office (text)
- doc_date (date)
- user_id (uuid, not null)
- created_at (timestamptz, default now())

Account

Sign in is required for secure cloud sync (Supabase Auth + Row Level Security).

Security notes 
This app uses:
- Supabase Auth (email + password)
- Row Level Security (RLS) on table: documents
Policy: user_id = auth.uid()
So each user can only see/edit/delete their own rows.
